Visual Studio Code is a free source-code editor made by Microsoft for Windows, Linux and macOS

Console is the infrastructure for console applications in Microsoft Windows. An instance of a Windows Console has a screen buffer and an input buffer. It allows console apps to run inside a window or in hardware text mode.

It is used to maintain or monitor the status of the network or computer.

This step on how to open it:

Right click on the screen and click on inspect and then in inspect click on the console section to see!!
